# Roboosk

Экосистема приложений для чтения, написания и создания миров.

## Структура

```
index.html              — Главная страница (лаунчер)
Roboosk-Library/        — Онлайн-библиотека (v1.12)
Roboosk-Wiki/           — Вики для писателей (Alpha 0.1)
Roboosk-Chat/           — Мессенджер (Alpha 0.2)
Roboosk-Reader/         — Читалка EPUB/FB2 (v1.93)
```
